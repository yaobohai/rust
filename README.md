## rust

Install rust

```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

New rust project

```
$ cargo new <project_name>
```

In vscode

```
// Command + P
ext install rust-lang.rust
```

Run

```
cargo run src/main.rs
```

Build

```
rustc src/main.rs
```
